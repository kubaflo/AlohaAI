# Top GitHub Repositories for AI Development

A curated collection of the most valuable open-source repositories for learning and building AI systems.

## LLM Frameworks & Orchestration

- **LangChain** (github.com/langchain-ai/langchain) — The most popular framework for building LLM applications with chains, agents, and retrieval
- **LlamaIndex** (github.com/run-llama/llama_index) — Data framework for connecting LLMs to external data sources with powerful retrieval capabilities
- **Semantic Kernel** (github.com/microsoft/semantic-kernel) — Microsoft's SDK for integrating LLMs into apps with plugins, planners, and memory
- **AutoGen** (github.com/microsoft/autogen) — Multi-agent conversation framework for building complex AI workflows
- **CrewAI** (github.com/crewAIInc/crewAI) — Framework for orchestrating role-playing AI agents working together

## Model Serving & Inference

- **vLLM** (github.com/vllm-project/vllm) — High-throughput LLM serving engine with PagedAttention
- **Ollama** (github.com/ollama/ollama) — Run LLMs locally with a simple CLI interface
- **llama.cpp** (github.com/ggml-org/llama.cpp) — Efficient C++ inference for LLMs on consumer hardware
- **ONNX Runtime** (github.com/microsoft/onnxruntime) — Cross-platform inference with hardware acceleration

## RAG & Vector Search

- **Chroma** (github.com/chroma-core/chroma) — Open-source embedding database for AI applications
- **Qdrant** (github.com/qdrant/qdrant) — High-performance vector similarity search engine
- **Weaviate** (github.com/weaviate/weaviate) — Vector database with built-in vectorization modules

## AI Safety & Evaluation

- **Guardrails AI** (github.com/guardrails-ai/guardrails) — Input/output guards for LLM applications
- **Garak** (github.com/NVIDIA/garak) — LLM vulnerability scanner for red teaming
- **DeepEval** (github.com/confident-ai/deepeval) — Unit testing framework for LLM outputs
